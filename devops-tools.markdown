## Active Projects

* [s3secrets - Fetches and decrypts files from S3 bucket using AWS KMS](https://github.com/UKHomeOffice/s3secrets)
* [kb8or - CI Tool for deploying with kubernetes](https://github.com/UKHomeOffice/kb8or)
* [vaultconf - Configures vault with a given list of policies and users](https://github.com/UKHomeOffice/vaultconf)
* [aws_usersync - A way of adding users and keys onto the OS by using AWS as the central place](https://github.com/UKHomeOffice/aws_usersync)
* [vault-sidekick - A vault sidekick container used for pulling secrets based on user policies](https://github.com/UKHomeOffice/vault-sidekick)
* [vcloud_ipsec-vpn - A way of creating VPN's between Vcloud Director Edge Gateways using Yaml](https://github.com/UKHomeOffice/vcloud_ipsec-vpn)
* [vcloud-start_stop - A way of stopping and starting VM's or entire VM's within VDC's to save money](https://github.com/UKHomeOffice/vcloud-start_stop)
* [vcloud_packager_images - A way of creating and pushing OVF Vmware Vapp images into Vcloud Director](https://github.com/UKHomeOffice/vcloud_packer_images)
